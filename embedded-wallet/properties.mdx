#### On finish authentication

Callback function called after either signup or login flows are finished `() => {}`

| Characteristics | Description    |
| --------------- | -------------- |
| Name            | `onFinishAuth` |
| Required        | **yes**        |

#### Production mode

When production mode is enabled, our production api will be called. `boolean?`

| Characteristics | Description |
| --------------- | ----------- |
| Name            | `isProd`    |
| Required        | no          |
| Default         | `true`      |
