## Usage

Copy the following code to the head of your website.

```JavaScript
<script src="https://widgets.lumxstudios.com"></script>

<script>
    widget.init({ clientId: <CLIENT_ID>, type: 'auth' })
</script>
```

You can also call **widget.init** on any event on your application such as on-click or on-submit.

## Events

To seamlessly incorporate the widget into your app, it's essential to monitor the emitted events. These events come in four distinct types.

#### INIT_IFRAME

When it has been called and loaded successfully.

#### USER_INFO

When user information is saved in the localStorage of the parent page. Contains the following user details.

```JavaScript
{
  username: string
  address:string
  avatar:string
  token:string
}
```

#### CLOSE_IFRAME

When widget is closed and removed from the page.

#### END_AUTH

When the authorization proccess is finished. User can then be redirected to the next page.
