---
title: "Customization"
description: "Transform your widgets with you unique style"
---

For further customization of the widgets, you can add the option object on the .init function. You can choose between the following keys:

#### theme

Defines the theme.

Default:
`dark`

#### primaryColor

Defines the color of the main button and the main shade color on the success screens both on checkout and auth.

Default:
`#6E05FF`

#### primaryButtonColor

Defines the color of the main button string.

Default:
`white`

#### secondaryColor

Defines the secondary shade on the success screens both on checkout and auth.

Default:
`#FA960399`

#### logo

The logo that appears on the success auth screen.

#### elementId

If the id of a div is provided, the widget will be appended to it.
